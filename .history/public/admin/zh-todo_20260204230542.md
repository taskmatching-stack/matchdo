# 後台重構（已切換至 MATCHDO 自研後台）

本清單已終止。舊版 Duralux 後台不再維護，並由自研後台取代：

## 新後台（自研）現況
- 儀表板：見 [admin/index.html](index.html)（共用標頭/側欄）。
- 分類管理：見 [admin/categories.html](categories.html)（API：GET/PUT /api/categories）。
- 共用部分檔：
	- 標頭 [admin/partials/header.html](partials/header.html)
	- 側欄 [admin/partials/sidebar.html](partials/sidebar.html)
	- 載入腳本 [admin/js/admin-common.js](js/admin-common.js)

### 進度更新（前台同步）
- 前台分類與提示詞：已改為從 `/api/categories` 即時載入，並設定 `Cache-Control: no-store`；首頁 `main.js` 亦使用 `fetch(..., { cache: 'no-store' })`，確保後台儲存後前台重新載入即可看到最新內容。

### 快速測試（免手動重建）
- 新增預設分類檔：`public/config/ai-categories.defaults.json`（空間施作/廣告牌設計/建模製作）。
- 開機自動引導：後端啟動時若 `ai_categories` 為空，會自動匯入預設分類。
- 手動一鍵匯入 API：`POST /api/categories/seed-defaults`（遇到清庫或測試時可立即重建）。
