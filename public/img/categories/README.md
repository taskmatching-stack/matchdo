# 分類預設圖片說明

這個資料夾存放各分類的預設封面圖片。

## 📸 圖片規格建議

- 尺寸：1200 x 800 px（3:2 比例）
- 格式：JPG 或 WebP
- 大小：< 500KB
- 風格：專業、清晰、符合分類主題

## 🎨 暫時方案

在正式圖片準備好之前，可以使用：

1. **Unsplash 免費圖片**
   - 網址：https://unsplash.com/
   - 搜尋關鍵字：interior design, carpentry, logo design 等
   
2. **Pexels 免費圖片**
   - 網址：https://www.pexels.com/
   - 高品質且可商用

3. **佔位圖服務**（臨時）
   - https://placehold.co/1200x800/0066CC/FFFFFF?text=室內設計
