## 三、admin 會員管理介面規劃（/admin/）

**目標**：在 `http://localhost:3000/admin/`（或實際 admin 路徑）新增「會員管理」相關介面，集中管理等級、點數與功能開關，方便後續調整數字與規則，**不在此階段寫程式**，僅規劃功能與資料。

### 3.1 建議新增的 admin 功能區塊

| 區塊 | 功能概要 | 說明 |
|------|----------|------|
| **會員等級／方案設定** | 列出四級（免費、10鎂、30鎂、60鎂）；可編輯每級名稱、月費、每月點數、顯示順序。 | 數字與名稱皆可後台改，不寫死。 |
| **點數規則** | 各「行為」對應點數（設計圖、描述詞、找廠商、製作方接案分類、對話介面跨國溝通翻譯每段）。 | 例如：設計圖 5 點/次、製作方接案分類 200 點/月、對話翻譯 1 點/段，皆可改。 |
| **功能權限矩陣** | 各等級對應：是否強制上靈感牆、製作方接案分類數量、是否顯示首頁對照圖、接案是否顯示於多語系。 | 以勾選/數字設定，方便日後加新等級或新權限。 |
| **用戶所屬等級與點數** | 查詢用戶目前方案、當月已用點數、剩餘點數、到期日；可手動調整方案或補點。 | 營運與客訴處理用。 |
| **點數流水／紀錄** | 查詢某用戶點數增減紀錄（發放、扣點、行為類型、時間）。 | 除錯與爭議查詢。 |

### 3.2 建議的「規則與數字」可調項目（後台表單或設定檔）

- **方案定義**：方案 key、顯示名稱、月費、每月點數、是否為免費方案。  
- **每方案權限**：  
  - 設計圖是否強制上靈感牆（是/否）  
  - 預設製作方接案分類數量（0／3／10／30 等，可改）  
  - 首頁是否可顯示作品與設計對照圖（是/否）  
  - 製作方接案是否顯示於多語系（跨境）頁面（是/否）  
- **點數消耗**：  
  - 設計圖一次 N 點  
  - 描述詞一次 N 點  
  - 找廠商一次 N 點（若僅免費方案扣點則加條件）  
  - 製作方接案分類（免費方案）N 點/月/分類  
  - 對話介面跨國溝通翻譯 1 點/段（或可改為 N 點/段）  

以上皆設計為「後台可改」，避免日後改方案要動程式。

### 3.3 與現有系統的對接（實作時再做）

- **Auth / 使用者**：會員等級需對應到使用者（例如 `profiles` 或 `subscriptions` 表）。  
- **點數**：需有「點數餘額」與「點數流水」表或欄位，並在設計圖、找廠商、對話介面跨國溝通翻譯等行為發生時扣點、寫入流水。  
- **靈感牆**：免費會員設計圖時強制 `show_on_homepage = true`；付費會員可選。  
- **製作方多語系／跨境顯示**：依方案決定製作方接案檔案是否在非預設語系（跨境）頁面顯示。  

實作時再依現有 DB 與路由設計細化 API 與畫面。
